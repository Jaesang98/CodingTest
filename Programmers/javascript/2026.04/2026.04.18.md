## 1단계 — 복습 풀이

### 문제: 숫자 문자열과 영단어 — 어제 푼 거 다시 풀기

**소요 시간:** 15분

어제 코드를 보지 않고 다시 풀었음. `replaceAll` 방식으로 접근.

### 풀이 흐름

| 버전     | 핵심 변화                                   |
| -------- | ------------------------------------------- |
| 처음     | `s`를 `const`로 선언했다가 재할당 오류 발생 |
| **최종** | `var`로 수정 + `replaceAll`로 전체 치환     |

---

### 처음 — `const` 재할당 오류

> `s = s.replaceAll(key, dNumbers[key])` 처럼 루프 안에서 계속 덮어써야 하는데, 변수를 `const`로 선언해서 에러 발생. 찾는 데 시간이 걸림.

```javascript
function solution(s) {
  var result = convertStringToNumber(s);
  return Number(result);
}

function convertStringToNumber(s) {
  var dNumbers = {
    zero: 0,
    one: 1,
    two: 2,
    three: 3,
    four: 4,
    five: 5,
    six: 6,
    seven: 7,
    eight: 8,
    nine: 9,
  };

  for (const key in dNumbers) {
    // ❌ const 이라 s 재할당 불가
    s = s.replaceAll(key, dNumbers[key]);
  }

  return s;
}
```

---

### 최종 — `const` → `var` 수정

```javascript
function solution(s) {
  var result = convertStringToNumber(s);
  return Number(result);
}

function convertStringToNumber(s) {
  var dNumbers = {
    zero: 0,
    one: 1,
    two: 2,
    three: 3,
    four: 4,
    five: 5,
    six: 6,
    seven: 7,
    eight: 8,
    nine: 9,
  };

  for (const key in dNumbers) {
    // ✅ s는 var이라 루프마다 덮어쓰기 가능
    s = s.replaceAll(key, dNumbers[key]);
  }

  return s;
}
```

## 1단계 문제 풀이 회고 — 숫자 문자열과 영단어 복습

### 문제: 콜라 문제 ([링크](https://school.programmers.co.kr/learn/courses/30/lessons/132267))

**소요 시간:** 총 약 40~50분 혼자 시도 → 해설 참고 후 재작성

### 풀이 흐름

| 버전    | 핵심 변화                                             |
| ------- | ----------------------------------------------------- |
| 1차     | 혼자 풀이 — `n > 1` 조건, 빈 병 처리 로직 오류        |
| **2차** | 해설 참고 후 재작성 — `n >= a` 조건으로 수정, 통과 ✅ |
| 3차     | 해설 보고 다시 작성 — 로직은 맞지만 변수명 불명확     |
| **4차** | 변수명 정리 완료 (GPT 도움) — 최종                    |

---

### 1차 시도 — 빈 병 처리 로직 오류

> `n > 1` 조건으로 루프를 돌렸는데 `blankCola` 함수 내부 로직이 잘못됨. 받은 콜라 수와 남은 빈 병 수를 혼용해 꼬임.

```javascript
function solution(a, b, n) {
  var limit = 0;
  while (n > 1) {
    limit += getCola(n, a) * b;
    n = blankCola(n, a) * b;
  }
  return limit;
}

function blankCola(n, a) {
  if (n % a === 0) return n / a;
  else return Math.floor(n / a) + (n % a);
}

function getCola(n, a) {
  if (n % a === 0) return n / a;
  else return Math.floor(n / a);
}
```

---

### 2차 시도 — 해설 참고 후 재작성 ✅ 통과

> 해설 보고 `while(n >= a)` 조건으로 수정. 함수도 역할에 맞게 분리.

```javascript
function solution(a, b, n) {
  var limit = 0;
  while (n >= a) {
    limit += getCola(n, a, b);
    n = blankCola(n, a) + getCola(n, a, b);
  }
  return limit;
}

function blankCola(n, a) {
  return n % a;
}

function getCola(n, a, b) {
  return Math.floor(n / a) * b;
}
```

---

### 3차 시도 — 해설 보고 다시 재작성

> 로직은 맞게 잡았지만 `test`라는 변수명이 의미 불명확.

```javascript
function solution(a, b, n) {
  var limit = 0;

  while (n >= a) {
    var test = n % a; // 못 낸 빈병
    n = Math.floor(n / a) * b; // 새로 받은 콜라
    limit += n;
    n = n + test; // 다음 라운드 총 병 수
  }

  return limit;
}
```

---

### 4차 시도 (최종) — 변수명 정리

```javascript
function solution(a, b, n) {
  var answer = 0;

  while (n >= a) {
    var rest = n % a; // 못 낸 빈병 (나머지)
    var newCola = Math.floor(n / a) * b; // 새로 받은 콜라

    answer += newCola;
    n = newCola + rest; // 다음 라운드 총 병 수
  }

  return answer;
}
```

## 오늘의 핵심 개념 정리

### `while` 문

```javascript
while (조건) {
  // 조건이 true인 동안 반복
  // 조건이 false가 되면 멈춤
}
```

> `while(n >= a)` — 빈 병이 `a`개 이상일 때만 교환 가능하므로 이 조건이 핵심.

---

### 콜라 문제 핵심 로직

```javascript
var rest = n % a; // 못 낸 빈병 (다음으로 이월)
var newCola = Math.floor(n / a) * b; // 빈병 내고 받은 새 콜라
n = newCola + rest; // 다음 라운드 총 병 수
```

| 변수      | 의미                                                   |
| --------- | ------------------------------------------------------ |
| `rest`    | 빈 병을 내고 남은 병 수 (`n % a`)                      |
| `newCola` | 빈 병을 내고 받은 새 콜라 수 (`Math.floor(n / a) * b`) |
| `n`       | 다음 라운드의 총 빈 병 수 (`newCola + rest`)           |
